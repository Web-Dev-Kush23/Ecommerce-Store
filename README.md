# 🛒 Ecommerce-Store (React)

A modern and responsive Ecommerce Store built using **React.js**.  
This project provides an attractive shopping UI with product listings, categories, cart functionality, and responsive design for all devices.

---

## 🚀 Features

- 🏠 Beautiful Home Page
- 🛍️ Product Listing Page
- 📦 Product Details Page
- 🛒 Shopping Cart Functionality
- ❤️ Wishlist Support
- 🔍 Product Search & Filters
- 📱 Fully Responsive Design
- ⚡ Fast Performance with React
- 🎨 Modern UI/UX

---

## 🛠️ Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **CSS3 / SCSS**
- **HTML5**
- **React Router**
- **Context API / Redux** *(if used)*
